User Interface

Access and manage training jobs through a simple, intuitive interface:

  1. Access Training Jobs: Navigate to LLM > Training > Jobs to view and manage all training jobs
  2. View Datasets: Access LLM > Training > Datasets to manage training data
  3. Monitor Status: Track job progress through the status bar and detailed metrics
  4. View Results: Access training results and metrics in the Results tab

Training Workflow

Create and manage training jobs with a streamlined workflow:

  1. Create Job: Click "Create" button to add a new training job
  2. Configure Job: Select provider, base model, and datasets
  3. Set Parameters: Configure hyperparameters for optimal training
  4. Submit Job: Click "Submit" to start the training process
  5. Monitor Progress: Track job status and metrics in real-time
