Get Started with Easy AI Chat

Follow these steps to set up and start chatting:

1

Install the Module

Add "Easy AI Chat" and its dependency "LLM Integration Base" to your Odoo environment.

2

Set Up Providers

Go to LLM > Configuration > Providers, create a new provider, and add your API key.

3

Fetch Models

Click "Fetch Models" in the provider form to import available AI models using your API key.

4

Start Chatting

Navigate to Chat in the Odoo menu, create a new thread, and ask AI anything!

Pro Tip: Get API keys from providers like OpenAI or Anthropic. Ensure your "LLM Integration Base" module is installed first.

How to Use Easy AI Chat

Once set up, chatting with AI is simple and flexible:

Start from Anywhere: Look for the "AI" button in any Odoo record's chat area to start a conversation.

Create a Thread: Click "New Chat" from the sidebar to start a fresh conversation.

Switch Models: Pick different AI providers or models from the thread header.

Integrate: Link chats to Odoo records via the chatter for context.

Select Tools: Choose specific tools for each thread to enable AI to perform actions in your Odoo system.
