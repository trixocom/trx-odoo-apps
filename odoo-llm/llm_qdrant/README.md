# LLM Qdrant Vector Database Integration

Qdrant vector database integration for high-performance semantic search.

## Features

- Qdrant vector storage
- High-performance similarity search
- Scalable vector operations
- Filtered search support

## Installation

1. Install the module
2. Requires llm_knowledge
3. Install Python dependency: `pip install qdrant-client`
4. Configure Qdrant connection

## Configuration

Set up Qdrant server connection in knowledge base settings:

- Host
- Port
- API key (if required)
- Collection name

## Requirements

- Odoo 18.0+
- llm_knowledge module
- qdrant-client Python package
- Qdrant server instance
