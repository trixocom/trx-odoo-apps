# LLM Knowledge Mistral Integration

Mistral AI embeddings and processing for knowledge base RAG.

## Features

- Mistral embedding models
- Efficient vector generation
- OCR capabilities via Mistral
- Optimized for French language

## Installation

1. Install the module
2. Requires llm_knowledge and llm_mistral
3. Configure Mistral embedding model

## Configuration

Select Mistral embedding model in knowledge base settings.

## Requirements

- Odoo 18.0+
- llm_knowledge module
- llm_mistral module
