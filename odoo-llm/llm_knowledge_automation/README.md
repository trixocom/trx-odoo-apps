# LLM Knowledge Automation

Automated knowledge collection and indexing for RAG-powered AI responses.

## Features

- Automatic knowledge chunk creation
- Scheduled indexing
- Automation rules integration
- Real-time knowledge updates

## Installation

1. Install the module
2. Requires llm_knowledge and base_automation
3. Configure automation rules

## Usage

Create automation rules to automatically index records as knowledge chunks for AI retrieval.

## Requirements

- Odoo 18.0+
- llm_knowledge module
- base_automation module
