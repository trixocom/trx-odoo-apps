# LLM Knowledge Llama Integration

LlamaIndex integration for advanced knowledge processing and RAG capabilities.

## Features

- LlamaIndex document processing
- Advanced chunking strategies
- Enhanced embedding support
- Query optimization

## Installation

1. Install the module
2. Requires llm_knowledge
3. Install Python dependencies: `pip install llama_index nltk`

## Configuration

Configure LlamaIndex-specific settings in knowledge base configuration.

## Requirements

- Odoo 18.0+
- llm_knowledge module
- llama_index Python package
- nltk Python package
